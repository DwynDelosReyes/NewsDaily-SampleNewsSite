This project is a frontend-only demonstration of a modern news website layout, designed to showcase UI/UX decisions, layout structure, and contemporary frontend development practices. All articles, headlines, and descriptions throughout the site are entirely fictional and exist solely to visualize how a real news platform might look and function.

Every news card on the website redirects to a sample article page, which intentionally breaks the fourth wall by explaining the project itself. No real news content is provided, and no backend or data fetching is implemented.

Key layout decisions include:

4. A right-hand column showcasing additional articles and an advertisement placeholder
